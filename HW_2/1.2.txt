1. Person 1 = 0..1 Teachers - 
	Человек есть Преподаватель. 
	Человек не обязан быть преподавателем, 
	Преподаватель должен быть человеком)

2. Person 1 = 0..1 Students - 
	Человек есть Студент. 
	Человек не обязан быть студентом, 
	Студент должен быть человеком.

3. Teachers 1 = 0..n Lessons - 
	Преподаватель ведёт занятие. 
	У Занятия должн быть преподаватель, 
	Преподаватель может проводить 0+ занятий.

4. Lessons 0..n = 1 Subject -
	Занятие по Учебному прдмету.
	Занятие проходит только по одному предмету,
	Занятий по предмету может быть сколько угодно.

5. Students 1..n = 1 Groups -
	Студент состоит в группе.
	Студент должен состоять в группе,
	В группе должны бвть студенты.

6. Groups 1 = 1..n Lessons -
	Группа имеет занятие.
	Одно занятие только у одной группы,
	Группа имеет хотя бы одно занятие или больше.

7. Marks - Семестровая оценка
	= 0..n Students
	= 0..n Lessons 
	Студент за занятие оценён оценкой Mark.
	У студентов может быть несколько оценок по занятиям.
	За занятие несколько студентов могут быть оценены.
